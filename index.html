📘 Pengertian OOP

OOP (Object Oriented Programming) adalah pendekatan pemrograman yang menjadikan objek sebagai pusat utama program. Konsep ini membantu program menjadi lebih terstruktur, rapi, dan mudah dikembangkan untuk skala besar.

🚗 Class dan Object

Class adalah cetakan untuk membuat objek. Sedangkan object adalah hasil nyata dari class tersebut. Contoh pada class Car, kita bisa membuat beberapa mobil dengan atribut dan fungsi yang sama.

🎓 Class Mahasiswa

Class Mahasiswa merepresentasikan data mahasiswa seperti nama, NIM, dan IPK. Dengan method tertentu, data dapat ditampilkan secara sistematis dan lebih profesional.

👤 Constructor

Constructor adalah method khusus yang otomatis dijalankan ketika objek dibuat. Fungsinya untuk menginisialisasi nilai awal dari atribut dalam class.

🛒 This Keyword & Product

Kata kunci this digunakan untuk menunjuk objek saat ini. Biasanya dipakai ketika nama parameter dan atribut sama, agar tidak terjadi konflik penamaan.

🛍 Class Produk

Class Produk digunakan untuk merepresentasikan data barang seperti id, nama, harga, stok, dan kategori. Dengan constructor parameterized, kita bisa membuat beberapa objek produk dengan nilai berbeda.
