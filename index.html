Pengertian Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program dibangun menggunakan class dan object yang berfungsi untuk merepresentasikan data serta perilaku. Tujuan utama OOP adalah membuat program lebih terstruktur, mudah dipahami, mudah dikembangkan, dan mudah dipelihara.

Konsep Dasar OOP

  • Class adalah cetak biru atau template untuk membuat object.
  • Object adalah hasil atau instance dari sebuah class.
  • Encapsulation adalah proses pembungkusan data dan method dalam satu class.
  • Inheritance adalah pewarisan sifat dari class induk ke class anak.
  • Polymorphism adalah kemampuan method memiliki banyak bentuk.
  • Abstraction adalah penyederhanaan kompleksitas dengan menyembunyikan detail tertentu.
