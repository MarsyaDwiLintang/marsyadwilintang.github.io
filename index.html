Pengertian OOP

Object Oriented Programming (OOP) merupakan metode pemrograman yang berorientasi pada objek. Dalam OOP, program dibangun menggunakan class dan object yang memiliki atribut serta method.

OOP bertujuan untuk membuat program lebih terstruktur, mudah dikembangkan, serta dapat digunakan kembali (reusability).

Contoh Implementasi Class pada Dart

class Car {
  String? model;
  String? color;

  void start() {
    print('Mobil dinyalakan');
  }

  void stop() {
    print('Mobil dimatikan');
  }
}

void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";

  print("Model Mobil: ${myCar.model}");
  print("Warna Mobil: ${myCar.color}");

  myCar.start();
  myCar.stop();
}

Kesimpulan

Dengan menggunakan konsep OOP, program menjadi lebih terorganisir karena setiap bagian memiliki tanggung jawab masing-masing. Konsep seperti encapsulation dan constructor sangat membantu dalam pengembangan aplikasi yang lebih kompleks.
