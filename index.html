📘 Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang menggunakan konsep objek dan class sebagai dasar pembuatan program.

OOP membantu membuat kode lebih terstruktur, reusable, modular, dan lebih mudah dikembangkan serta dipelihara.

Konsep utama OOP meliputi: Class, Object, Constructor, Encapsulation, Inheritance, dan Polymorphism.
