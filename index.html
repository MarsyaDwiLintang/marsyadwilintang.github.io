Pengertian Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program dibuat dengan menggunakan class dan object yang berfungsi untuk mengatur data serta perilaku. Konsep ini membantu program menjadi lebih terstruktur, rapi, dan mudah dikembangkan.

Konsep Dasar OOP

  • Class → Template atau cetak biru dalam membuat object.
  • Object → Instance dari class.
  • Encapsulation → Pembungkusan data dan method dalam satu class.
  • Inheritance → Pewarisan sifat dari class induk ke class anak.
  • Polymorphism → Satu method memiliki banyak bentuk.
  • Abstraction → Penyembunyian detail implementasi.
