Pengertian Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program dibangun menggunakan class dan object yang saling berinteraksi.

Tujuan OOP adalah membuat kode lebih terstruktur, mudah dipahami, dapat digunakan kembali (reusable), dan lebih mudah dikembangkan.

Konsep Dasar OOP

1. Class adalah blueprint atau cetakan untuk membuat object. Class berisi atribut (data) dan method (fungsi).

2. Object adalah instance atau hasil nyata dari sebuah class. Object memiliki nilai pada atribut dan dapat menjalankan method.

3. Atribut (Property) adalah data atau karakteristik yang dimiliki oleh sebuah object.

4. Method adalah fungsi atau perilaku yang dapat dilakukan oleh object.

5. Constructor adalah method khusus yang dipanggil saat object dibuat untuk menginisialisasi nilai awal atribut.

6. Encapsulation adalah teknik menyembunyikan data dengan membuat atribut private dan mengaksesnya melalui getter dan setter.

7. Inheritance adalah pewarisan sifat dari satu class ke class lain untuk mengurangi duplikasi kode.

8. Polymorphism adalah kemampuan suatu method memiliki banyak bentuk atau perilaku berbeda.
