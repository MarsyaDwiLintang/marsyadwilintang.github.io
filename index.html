Pengertian Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program disusun berdasarkan class dan object yang saling berinteraksi.

OOP bertujuan untuk membuat kode lebih terstruktur, mudah dipahami, dapat digunakan kembali (reusable), serta mudah dikembangkan.

Konsep Dasar OOP

1. Class
Class adalah blueprint atau cetakan untuk membuat objek. Class berisi atribut (data) dan method (fungsi).

2. Object
Object adalah instance atau hasil nyata dari sebuah class. Object memiliki data dan dapat menjalankan method yang ada di dalam class.

3. Constructor
Constructor adalah method khusus yang dipanggil saat objek dibuat. Constructor digunakan untuk menginisialisasi nilai atribut.

4. Encapsulation
Encapsulation adalah teknik menyembunyikan data agar tidak bisa diakses langsung dari luar class. Biasanya menggunakan private variable dan method getter/setter.

5. Abstraction
Abstraction adalah penyederhanaan kompleksitas dengan hanya menampilkan bagian penting dari suatu objek.

6. Inheritance
Inheritance adalah pewarisan sifat dari satu class ke class lain agar kode bisa digunakan kembali.

7. Polymorphism
Polymorphism memungkinkan satu method memiliki banyak bentuk atau perilaku berbeda.
