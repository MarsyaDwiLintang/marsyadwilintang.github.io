Tugas Pemrograman Berbasis Objek (OOP) - Dart

Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program disusun menggunakan class dan object untuk memodelkan suatu sistem.

Konsep Dasar OOP

  • Class → Blueprint atau cetakan untuk membuat objek
  • Object → Hasil dari class
  • Encapsulation → Pembungkusan data
  • Inheritance → Pewarisan sifat
  • Polymorphism → Banyak bentuk
  • Abstraction → Penyederhanaan sistem

Contoh Kode Dart

class Mahasiswa {
  String nama;
  String nim;

  Mahasiswa(this.nama, this.nim);

  void tampilkanData() {
    print("Nama: $nama");
    print("NIM: $nim");
  }
}

void main() {
  var mhs = Mahasiswa("Marsya Dwi Lintang", "251410028");
  mhs.tampilkanData();
}
