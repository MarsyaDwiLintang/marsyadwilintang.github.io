Konsep Dasar Object Oriented Programming (OOP)

1. Class
Class adalah cetakan atau blueprint untuk membuat objek. Di dalam class terdapat atribut (variabel) dan method (fungsi). Class mendefinisikan bagaimana sebuah objek akan dibuat dan bekerja.

2. Object
Object adalah hasil nyata (instance) dari sebuah class. Object memiliki nilai pada atribut dan dapat menjalankan method yang sudah didefinisikan di dalam class.

3. Atribut (Property)
Atribut adalah data atau karakteristik yang dimiliki oleh sebuah objek. Contohnya pada class Car terdapat atribut model dan color.

4. Method
Method adalah fungsi atau perilaku yang dimiliki oleh sebuah class. Method menentukan apa yang bisa dilakukan oleh objek. Contohnya start() dan stop().

5. Constructor
Constructor adalah method khusus yang otomatis dipanggil saat objek dibuat. Constructor digunakan untuk menginisialisasi nilai awal atribut.

6. Encapsulation
Encapsulation adalah proses menyembunyikan data agar tidak bisa diakses langsung dari luar class. Biasanya menggunakan private variable dan getter/setter.

7. Inheritance
Inheritance adalah pewarisan sifat dari satu class ke class lain. Dengan inheritance, kita bisa menggunakan kembali kode yang sudah ada.

8. Polymorphism
Polymorphism adalah kemampuan suatu method memiliki banyak bentuk atau perilaku berbeda tergantung konteks penggunaannya.
