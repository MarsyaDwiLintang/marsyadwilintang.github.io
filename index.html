Pengertian Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program disusun menggunakan class dan object yang merepresentasikan data dan perilaku.

OOP bertujuan untuk membuat program lebih terstruktur, mudah dikembangkan, dan mudah dipelihara.

Konsep Dasar OOP

  • Class → Cetak biru (blueprint) untuk membuat object.
  • Object → Instance dari class.
  • Encapsulation → Pembungkusan data dan method dalam satu class.
  • Inheritance → Pewarisan sifat dari class induk ke class anak.
  • Polymorphism → Kemampuan method memiliki banyak bentuk.
  • Abstraction → Penyederhanaan kompleksitas dengan menyembunyikan detail tertentu.
