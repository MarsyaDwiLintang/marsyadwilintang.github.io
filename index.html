Ringkasan Materi OOP (Tanpa 7–9)

Poin penting: pengantar OOP, konsep objek & kelas, serta dasar class di Dart.

A. Pengantar OOP

  • OOP = paradigma yang fokus pada objek sebagai unit utama.
  • Objek memiliki state (data) dan behavior (perilaku/metode).
  • Tujuan: program lebih modular, reusable, dan mudah dirawat.

B. Objek, Kelas, Atribut, Metode

  • Kelas = blueprint/cetakan; Objek = instance nyata.
  • Atribut = variabel pada class (contoh: warna/model).
  • Metode = fungsi pada class (contoh: start()/stop()).

C. Imperatif vs OOP

  • Imperatif: fokus langkah prosedural.
  • OOP: fokus interaksi objek untuk menyelesaikan masalah.

D. OOP di Dart

  • Dart mendukung OOP dan sering dipakai di Flutter.
  • OOP membantu struktur kode ketika aplikasi makin kompleks.
