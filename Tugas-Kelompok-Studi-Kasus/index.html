Permasalahan

Sebuah platform kursus online memiliki banyak siswa, instruktur, dan kelas yang harus dikelola. Jika pencatatan dilakukan secara manual, akan sulit mengetahui siswa yang terdaftar, instruktur yang mengajar, dan kursus yang tersedia. Oleh karena itu, diperlukan sistem yang terstruktur agar pengelolaan data menjadi lebih mudah dan efisien.

Mengapa menggunakan OOP?

Object Oriented Programming (OOP) digunakan karena sistem terdiri dari beberapa objek yang saling berhubungan, yaitu siswa, instruktur, dan kursus. Dengan OOP, data dan fungsi dapat dikelompokkan ke dalam class yang sesuai sehingga program menjadi lebih rapi, mudah dipahami, dan mudah dikembangkan.

Bagaimana cara penyelesaiannya?

Penyelesaian dilakukan dengan membuat beberapa class sesuai objek yang ada dalam sistem:

  • Class Student untuk menyimpan data siswa.

  • Class Instructor untuk menyimpan data instruktur.

  • Class Course untuk menyimpan data kursus.

  • Menggunakan Enkapsulasi untuk melindungi data tertentu.

  • Menggunakan Abstraksi untuk mendefinisikan proses pendaftaran kursus.

  • Menggunakan Inheritance agar class turunan dapat mewarisi atribut dan method dari class induk.

  • Menggunakan Polimorfisme agar method yang sama dapat menghasilkan perilaku yang berbeda

DART SOURCE CODE

// IMPLEMENTASI OOP DENGAN DART

// Class untuk menyimpan data siswa
class Student {
  String name;
  String email;

  Student(this.name, this.email);

  void displayInfo() {
    print("Nama Siswa : $name");
    print("Email       : $email");
  }
}

// Inheritance dan Polimorfisme
class PremiumStudent extends Student {
  PremiumStudent(String name, String email)
      : super(name, email);

  @override
  void displayInfo() {
    print("=== Siswa Premium ===");
    print("Nama Siswa : $name");
    print("Email       : $email");
  }
}

// Class instruktur
class Instructor {
  String name;

  Instructor(this.name);

  void displayInfo() {
    print("Instruktur : $name");
  }
}

// Enkapsulasi
class Course {
  String _courseName;

  Course(this._courseName);

  String get courseName => _courseName;

  void setCourseName(String name) {
    _courseName = name;
  }

  void displayInfo() {
    print("Kursus : $_courseName");
  }
}

// Abstraksi
abstract class Enrollment {
  void register();
}

// Implementasi abstraksi
class CourseEnrollment implements Enrollment {
  Student student;
  Course course;
  Instructor instructor;

  CourseEnrollment(
      this.student,
      this.course,
      this.instructor);

  @override
  void register() {
    print(
        "${student.name} berhasil mendaftar kursus ${course.courseName}");
    print(
        "Instruktur Kursus : ${instructor.name}");
  }
}

void main() {

  Student student =
      Student(
          "Marsya",
          "marsya@gmail.com");

  Student premiumStudent =
      PremiumStudent(
          "Putri",
          "putri@gmail.com");

  Instructor instructor =
      Instructor("Pak Alex");

  Course course =
      Course("Pemrograman Dart");

  student.displayInfo();

  print("");

  premiumStudent.displayInfo();

  print("");

  instructor.displayInfo();

  course.displayInfo();

  print("");

  CourseEnrollment enrollment =
      CourseEnrollment(
          student,
          course,
          instructor);

  enrollment.register();
}

Sistem Kursus Online Sederhana merupakan program berbasis Dart yang dibuat untuk mengelola data siswa, instruktur, dan kursus dengan menerapkan konsep Object Oriented Programming (OOP). Saat program dijalankan, sistem akan membuat data siswa, siswa premium, instruktur, dan kursus berdasarkan objek yang telah dibuat. Selanjutnya sistem menampilkan informasi siswa, instruktur, dan kursus yang tersedia. Output yang dihasilkan berupa informasi siswa, informasi instruktur, informasi kursus, serta status pendaftaran kursus yang berhasil dilakukan. Melalui program ini, konsep Objek dan Class, Enkapsulasi, Abstraksi, Inheritance, dan Polimorfisme diterapkan secara terintegrasi dalam satu Sistem Kursus Online Sederhana
