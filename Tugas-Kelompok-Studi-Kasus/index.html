Permasalahan

Sebuah platform kursus online memiliki banyak siswa, instruktur, dan kelas yang harus dikelola. Jika pencatatan dilakukan secara manual, akan sulit mengetahui siswa yang terdaftar, instruktur yang mengajar, dan kursus yang tersedia. Oleh karena itu, diperlukan sistem yang terstruktur agar pengelolaan data menjadi lebih mudah dan efisien.

Mengapa menggunakan OOP?

Object Oriented Programming (OOP) digunakan karena sistem terdiri dari beberapa objek yang saling berhubungan, yaitu siswa, instruktur, dan kursus. Dengan OOP, data dan fungsi dapat dikelompokkan ke dalam class yang sesuai sehingga program menjadi lebih rapi, mudah dipahami, dan mudah dikembangkan.

Bagaimana cara penyelesaiannya?

Penyelesaian dilakukan dengan membuat beberapa class sesuai objek yang ada dalam sistem:

  • Class Student untuk menyimpan data siswa.

  • Class Instructor untuk menyimpan data instruktur.

  • Class Course untuk menyimpan data kursus.

  • Menggunakan Enkapsulasi untuk melindungi data tertentu.

  • Menggunakan Abstraksi untuk mendefinisikan proses pendaftaran kursus.

  • Menggunakan Inheritance agar class turunan dapat mewarisi atribut dan method dari class induk.

  • Menggunakan Polimorfisme agar method yang sama dapat menghasilkan perilaku yang berbeda

// Class untuk menyimpan data siswa
class Student {
  String name;
  String email;
  List enrolledCourses = [];

  Student(this.name, this.email);

  // Batas maksimal kursus siswa biasa
  int get maxCourseLimit => 1;

  // Mengecek apakah siswa sudah terdaftar pada kursus
  bool isEnrolled(Course course) {
    return enrolledCourses.contains(course);
  }

  // Mengecek apakah siswa masih bisa mengambil kursus
  bool canTakeMoreCourses() {
    return enrolledCourses.length < maxCourseLimit;
  }

  // Menambahkan kursus ke daftar siswa
  void addCourse(Course course) {
    enrolledCourses.add(course);
  }

  // Menampilkan informasi siswa
  void displayInfo() {
    print("Nama : $name");
    print("Email : $email");
    print("Jumlah Kursus : ${enrolledCourses.length}/$maxCourseLimit");
  }
}

// Inheritance dan Polymorphism
class PremiumStudent extends Student {
  PremiumStudent(String name, String email)
      : super(name, email);

  @override
  int get maxCourseLimit => 5;
}

// Class untuk menyimpan data instruktur
class Instructor {
  String name;

  Instructor(this.name);

  void displayInfo() {
    print("Instruktur : $name");
  }
}

// Encapsulation
class Course {
  String _courseName;

  Course(this._courseName);

  String get courseName => _courseName;

  void setCourseName(String name) {
    if (name.trim().isNotEmpty) {
      _courseName = name;
    }
  }

  void displayInfo() {
    print("Kursus : $_courseName");
  }
}

// Abstraction
abstract class Enrollment {
  bool register();
}

// Class untuk proses pendaftaran kursus
class CourseEnrollment implements Enrollment {
  Student student;
  Course course;
  Instructor instructor;

  CourseEnrollment(
      this.student,
      this.course,
      this.instructor);

  @override
  bool register() {

    // Cek pendaftaran ganda
    if (student.isEnrolled(course)) {
      print("${student.name} gagal mendaftar ${course.courseName} (sudah terdaftar)");
      return false;
    }

    // Cek batas maksimal kursus
    if (!student.canTakeMoreCourses()) {
      print("${student.name} gagal mendaftar ${course.courseName} (batas kursus tercapai)");
      return false;
    }

    // Menambahkan kursus ke daftar siswa
    student.addCourse(course);

    print("${student.name} berhasil mendaftar ${course.courseName}");
    return true;
  }
}

Penjelasan Penggunaan Aplikasi

Sistem Kursus Online Sederhana merupakan program berbasis Dart yang dibuat untuk mengelola data siswa, instruktur, dan kursus dengan menerapkan konsep Object Oriented Programming (OOP).

Program dijalankan melalui fungsi main() yang menjadi titik awal eksekusi program. Saat program dijalankan, sistem akan membuat data siswa, siswa premium, instruktur, dan kursus berdasarkan objek yang telah dibuat.

Selanjutnya sistem menampilkan informasi siswa, instruktur, dan kursus yang tersedia. Setelah itu dilakukan proses pendaftaran kursus melalui class CourseEnrollment yang menghubungkan siswa dengan kursus dan instruktur.

Output yang dihasilkan berupa informasi siswa, informasi instruktur, informasi kursus, serta status pendaftaran kursus yang berhasil dilakukan.

Melalui program ini, konsep Objek dan Class, Enkapsulasi, Abstraksi, Inheritance, dan Polimorfisme diterapkan secara terintegrasi dalam satu Sistem Kursus Online Sederhana.
