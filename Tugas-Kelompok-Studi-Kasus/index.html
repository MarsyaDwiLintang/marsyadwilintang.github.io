Permasalahan

Sebuah platform kursus online memiliki banyak siswa, instruktur, dan kelas yang harus dikelola. Jika pencatatan dilakukan secara manual, akan sulit mengetahui siswa yang terdaftar, instruktur yang mengajar, dan kursus yang tersedia. Oleh karena itu, diperlukan sistem yang terstruktur agar pengelolaan data menjadi lebih mudah dan efisien.

Mengapa menggunakan OOP?

Object Oriented Programming (OOP) digunakan karena sistem terdiri dari beberapa objek yang saling berhubungan, yaitu siswa, instruktur, dan kursus. Dengan OOP, data dan fungsi dapat dikelompokkan ke dalam class yang sesuai sehingga program menjadi lebih rapi, mudah dipahami, dan mudah dikembangkan.

Bagaimana cara penyelesaiannya?

Penyelesaian dilakukan dengan membuat beberapa class sesuai objek yang ada dalam sistem:

  • Class Student untuk menyimpan data siswa.

  • Class Instructor untuk menyimpan data instruktur.

  • Class Course untuk menyimpan data kursus.

  • Menggunakan Enkapsulasi untuk melindungi data tertentu.

  • Menggunakan Abstraksi untuk mendefinisikan proses pendaftaran kursus.

  • Menggunakan Inheritance agar class turunan dapat mewarisi atribut dan method dari class induk.

  • Menggunakan Polimorfisme agar method yang sama dapat menghasilkan perilaku yang berbeda

01. Objek & Class

Pada sistem ini terdapat beberapa objek utama yaitu Student, Instructor, dan Course. Class Student digunakan untuk menyimpan data siswa berupa nama dan email. Objek student dibuat dari class Student dan dapat menampilkan informasi siswa

DART SOURCE CODE

class Student {
  String name;
  String email;

  Student(this.name, this.email);

  void displayInfo() {
    print("Nama: $name");
    print("Email: $email");
  }
}

void main() {
  Student student =
      Student("putri marsya", "putrimarsya1414@gmail.com");

  student.displayInfo();
}

02. Enkapsulasi

Enkapsulasi digunakan untuk melindungi data agar tidak dapat diakses secara langsung dari luar class. Variabel _courseName bersifat private sehingga tidak dapat diakses langsung dari luar class. Data hanya dapat diakses melalui getter dan setter

DART SOURCE CODE

class Course {
  String _courseName;

  Course(this._courseName);

  String get courseName => _courseName;

  void setCourseName(String name) {
    _courseName = name;
  }
}

void main() {
  Course course = Course("Pemrograman Dart");

  print(course.courseName);

  course.setCourseName("Flutter Dasar");

  print(course.courseName);
}

03. Abstraksi

Abstraksi digunakan untuk menyembunyikan detail implementasi dan hanya menampilkan fungsi yang diperlukan. Class Enrollment merupakan abstract class yang berisi method register(). Class CourseEnrollment mengimplementasikan method tersebut.

DART SOURCE CODE

abstract class Enrollment {
  void register();
}

class CourseEnrollment implements Enrollment {
  @override
  void register() {
    print("Siswa berhasil mendaftar kursus");
  }
}

void main() {
  CourseEnrollment enrollment =
      CourseEnrollment();

  enrollment.register();
}

04. Inheritance

Inheritance digunakan agar suatu class dapat mewarisi atribut dan method dari class lain. Class PremiumStudent mewarisi seluruh atribut dan method dari class Student sehingga tidak perlu menulis ulang kode yang sama.

DART SOURCE CODE

class Student {
  String name;

  Student(this.name);

  void displayInfo() {
    print("Nama Siswa: $name");
  }
}

class PremiumStudent extends Student {
  PremiumStudent(String name)
      : super(name);
}

void main() {
  PremiumStudent student =
      PremiumStudent("Putri Marsya");

  student.displayInfo();
}

05. Polimorfisme

Polimorfisme memungkinkan method yang sama memiliki perilaku yang berbeda. Method displayInfo() pada PremiumStudent menimpa method yang ada pada Student sehingga menghasilkan output yang berbeda.

DART SOURCE CODE

class Student {
  String name;

  Student(this.name);

  void displayInfo() {
    print("Siswa: $name");
  }
}

class PremiumStudent extends Student {
  PremiumStudent(String name)
      : super(name);

  @override
  void displayInfo() {
    print("Siswa Premium: $name");
  }
}

void main() {
  Student student1 =
      Student("Marsya");

  Student student2 =
      PremiumStudent("Putri");

  student1.displayInfo();
  student2.displayInfo();
}
